# 🏏 AMAWAN PREMIUM LEAGUE (APL) SCOREBOARD

## Overview

APL Scoreboard is a cricket scoring web application built using HTML, CSS, and JavaScript. It is designed for local cricket tournaments and supports live score updates, batting statistics, bowling statistics, match results, and match history storage.

---

## Features

### Match Setup

* Team A vs Team B
* Team Logo Upload
* Toss Winner
* Target Score

### Live Scoreboard

* Runs & Wickets
* 12 Overs Match Format
* Current Run Rate (CRR)
* Required Run Rate (RRR)

### Batting

* Striker & Non-Striker
* Individual Batting Scores
* Balls Faced Tracking

### Bowling

* Bowler Name
* Bowler Overs
* Bowler Runs Conceded
* Bowler Wickets

### Extras

* Wide
* No Ball
* Bye
* Leg Bye

### Match Tracking

* Fall of Wickets (FOW)
* Match Result
* Man of the Match

### Storage

* Save Match to Browser Local Storage
* Retrieve Saved Match Data

### User Interface

* Mobile Responsive Design
* Clean Scoreboard Layout

## Installation

1. Download all project files.
2. Place the files in the same folder.
3. Open `index.html` in any modern browser.

No server installation is required.

---

## Run on Mobile

### Using Acode

1. Install Acode App.
2. Create a new folder.
3. Paste:

   * index.html
   * style.css
   * script.js
4. Open `index.html`.
5. Tap Preview.

### Using Spck Editor

1. Install Spck Editor.
2. Create a project.
3. Add all files.
4. Run the project.
